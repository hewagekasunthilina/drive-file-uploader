
-----------Instructions to run the project------------------------------

* First of all, Xampp control panel should be available in your machine.
* Launch Xampp Control Panel.
* Apache Server should be turn on using Xampp Control Panel.
* Apache server should be running on port 80. If not change your server to port 80
* Copy the project to the root folder of Apache
* Finally run "localhost:80/drive-file-uploader/index.php" on your browser
